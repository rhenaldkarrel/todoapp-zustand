# Todo List Application

Simple Todo List/Task Tracker application built with React.js + TypeScript and Zustand state manager.

## Features

- Add new todo
- Update todo
- Delete todo
- Mark todo as done

## Libraries Used

- Zustand
- Headless UI
- Tailwind CSS

## How to start the app

Install dependencies.

```bash
pnpm install
```

Run development.

```bash
pnpm dev
```
